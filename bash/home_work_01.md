1) Написать скрипт для инициализации рабочей директории для проекта на python; необходимые действия
- имя проекта (название корневой папки) задается через позиционный аргумент
- структура каталогов
```
.
├── .gitignore
├── .venv
├── README.md
├── docs
├── pyproject.toml
├── requirements.txt
├── src
│   ├── __init__.py
│   └── main.py
└── tests
```
где `.venv` - виртуальное окружение, создается и активируется командами
```
python3 -m venv .venv
source .venv/bin/activate
```
2) Написать скрипт, который будет проверять минимальную работоспособность сайта - адрес сайта должен быть передан скрипту как аргумент. Сайт считается работающим, если он отдает код 200
3) Написать скрипт, который будет проверять состояние указанной в аргументе службы и перезапускать ее, если она остановлена. Например, сервис nginx
